Gemini System

Welcome to our Gemini System !!

Description: Gemini System is the system which collaborated with the OCS System for providing these activities as follow:

      - As an Astronomer:

           - Create science plan

           - Test science plan

           - Submit science plan

      - As an Science Observer:

           - Validate science plan

           - Create observing program

User Journey:

      1.) Astronomer creates a science plan (STATUS: null) -> test science plan (STATUS: null -> RUNNING)

      2.) Science observer validates the science plan (STATUS: RUNNING -> COMPLETE)

      3.) Astronomer submits the science plan (STATUS: COMPLETE -> SUBMITTED)

      4.) Science observer create an observing program from the submitted science plan

      STATUS = [null, RUNNING, COMPLETE, SUBMITTED]
